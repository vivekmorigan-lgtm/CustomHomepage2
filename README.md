# Vite + React Clock Component

This project is a simple clock component built with Vite and React. It displays the current time and date, along with a moving dot that follows the mouse cursor.

## Features

- Displays current time and date
- Moving dot that follows the mouse cursor
- Responsive design
- Built with Vite for fast development

## Getting Started

To get started with this project, follow the instructions below:

### Copy this link

[https://homepage015.pages.dev/](https://homepage015.pages.dev/)

### Add as a new tab homepage

1. Open your web browser.
2. Go to the settings or preferences menu.
3. Look for the option to set a custom homepage or new tab page.
4. Enter the URL: `https://homepage015.pages.dev/`
5. Save your changes.
6. Open a new tab to see your custom homepage in action!

#### OR

1. Open your web browser.
2. Open chromewebstore
3. Search "Custom New Tab"
4. Add to chrome
5. Open and add link to new tab to see your custom homepage in action!
